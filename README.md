# ⛏️ Mineras BI Chile 2026
Proyecto integral de Business Intelligence para el análisis de producción y eficiencia financiera en la minería chilena.

## 🛠️ Tecnologías Utilizadas
- **Python**: Extracción e ingesta de datos mediante scripts automatizados.
- **MySQL**: Almacenamiento y modelado de datos en esquema estrella.
- **Power BI**: Visualización interactiva y análisis de KPIs (EBITDA vs Producción).

## 📂 Contenido del Repositorio
- `/chile-mineria-pbi`: Scripts de Python para el proceso ETL.
- `mineras_sql`: Script para recrear la base de datos en MySQL.
- `PRODUCCION_MINERA.pbix`: Reporte interactivo final.
